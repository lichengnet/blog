当我给老板展示我最终修复了那个 bug 时

当项目经理走进办公室

当我在为产品编码时

当我的正则表达式返回我想要的结果时

虽然我没有对代码进行测试，但是在产品环境中运行良好时

当我首次使用一个新的 CSS

当我周末休息，而其他所有人还在努力修复 bug 时

当我回头去开发我那些没有注释的代码时

当他们问我是否测试时
